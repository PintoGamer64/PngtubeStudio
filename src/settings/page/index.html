Asistente de configuración de PngtubeStudio

Bienvenido al asistente de configuracion de PngtubeStudio.

Este asistente creara la configuracion necesaria para ejecutar el programa ¿Quieres ejecutar la herramienta?

¿Que hace este asistente?

Este asistente creara directorios y archivos que el programa necesita para guardar configuraciones, modelos y demas elementos que requieran ser leidas en su PC creadas por el programa (este asistente se ejecutara si hay errores en directorios o archivos esenciales para el correcto funcionamiento del programa)
